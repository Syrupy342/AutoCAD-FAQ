AutoCAD FAQ

Font is not loading correctly in AutoCAD. How do I fix it?

To add a font path in AutoCAD, use the Supported File Search Path in the Options dialog box:

  1. Type OPTIONS on the command line.
  2. In the Files tab, open the tree for Supported File Search Path.
  3. Click Add.
  4. Click Browse.
  5. Navigate to the folder where the fonts are located (G:\Engineering\CAD Standards\7.0 SUPPORT\FONTS).
  6. Click OK.
  7. Click Apply.
  8. Close and reopen AutoCAD.
